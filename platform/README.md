# platform

Comprehensive platform for building web-based client applications. Provides:

* Touch input
* Keyboard input
* Vector rendering
* Raster rendering
* Animation
* Audio
* Browser APIs
    * Alert
    * Clipboard
    * Cursor
    * Fetch
    * Frame
    * Fullscreen
    * Image
    * Keyboard
    * Local storage
    * Location
    * Mobile detection
    * Performance
    * Size
    * Sound
    * Touch
    * Websocket
    * Worker
